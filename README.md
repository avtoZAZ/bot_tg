# bot_tg

Telegram Video Bot — бот для перегляду відео через Telegram з підтримкою преміум підписки, реферальної системи, промокодів та аналітики.

## Можливості

- 🎬 Пошук та перегляд відео за кодом
- ⭐ Преміум підписка (тижнева / місячна / річна)
- 📜 Історія переглядів та обране
- ⭐ Рейтинг відео (1–5 зірок)
- 📺 Підписка на серіали та сповіщення про нові епізоди
- 🎫 Промокоди для безкоштовного преміуму
- 👥 Реферальна система з бонусами
- 📢 Масова розсилка (для адміністраторів)
- 📊 Аналітика та статистика
- 🌐 Підтримка трьох мов: English, Русский, Українська

## Вимоги

- Python 3.10+
- Залежності зі списку `telegram_bot_2.1/requirements.txt`

## Встановлення та запуск

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/avtoZAZ/bot_tg.git
   cd bot_tg/telegram_bot_2.1
   ```

2. **Встановіть залежності:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Налаштуйте змінні середовища:**
   ```bash
   cp ../.env.example .env
   # Відредагуйте .env та вкажіть BOT_TOKEN та ADMIN_ID
   ```

4. **Запустіть бота:**
   ```bash
   python bot.py
   ```

## Налаштування (.env)

Скопіюйте `.env.example` як `.env` та заповніть необхідні поля:

| Змінна | Опис | За замовчуванням |
|--------|------|-----------------|
| `BOT_TOKEN` | Токен Telegram бота (від @BotFather) | — |
| `ADMIN_ID` | Telegram ID головного адміністратора | — |
| `DATABASE_URL` | URL підключення до бази даних | `sqlite+aiosqlite:///bot_data.db` |
| `ADS_ENABLED` | Увімкнути рекламу | `True` |
| `REFERRAL_ENABLED` | Увімкнути реферальну систему | `True` |

Повний список змінних дивіться у `.env.example`.
